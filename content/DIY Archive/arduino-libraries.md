This page provides an overview of the libraries I've used in my projects. Where I've modified a library, you can download the modified version. Otherwise, it links to the original source (or possibly a newer version).

- **VirtualWire v1.9** - Send/receive data using cheap hardware. [Download PDF manual](#).
- **AVVISO** - Send alarms to Prowl!
- **DHT11/22 Library** - Temperature & humidity sensor.
- **Arduino Manager** - Control Arduino from iPhone/iPad. You need to buy the app to use it.
- **RemoteSwitch Library** - Remote control switches using a 434MHz transmitter. [Visit this topic](#) too.
- **Jeelib** - Library from Jeelabs.org. Contains the RF12 library for the RFM12B transceivers.
