I've created a mount for the Sensus 620 watermeter to mount a  [LJ18A3-8-Z/BX-5V](https://nl.aliexpress.com/item/32844213899.html) proximity sensor on top of it. The sensor is connected using the schematic which can be found [here](https://www.ztatz.nl/p1-monitor-watermeter/). Instead of using a Raspberry Pi, I use a Wemos D1 with Tasmota firmware.

The [3D print design](https://www.thingiverse.com/thing:4489072) can be found on Thingiverse.
